# Seletores

Seletor Função Exemplo CSS
* Seleciona todas as tags HTML *{color: violet}
x Seleciona todas tags x x {color: violet}
x y Seleciona todas as tags z que estiver contido num elemento x x y{color: violet}
x > y Seletor de elemento-filho: Seleciona qualquer elemento y descendente direto de uma tag x x > y{color: violet}
x + y Seleciona o elemento y que estiver após um elemento x x + y{color: violet}
.class Seleciona o elemento com a class "classe" .classe{color: violet}
#destaque Seleciona o elemento com id igual a "destaque" #destaque{color: violet}
x,y Seleciona o elemento x e y x,y{color: violet}
x[atributo] Seleciona o elemento x que contenha o atributo indicado x[title]{color: violet}
x[atributo="valor"] Seleciona o elemento x que contenha o atributo e o valor indicado input[type=number]{color: violet}
