# Seletores

Seletor Função Exemplo CSS
* Seleciona todas as tags HTML *{color: violet}
x Seleciona todas tags x x {color: violet}
x y Seleciona todas as tags z que estiver contido num elemento x x y{color: violet}
x > y Seletor de elemento-filho: Seleciona qualquer elemento y descendente direto de uma tag x x > y{color: violet}
x + y Seleciona o elemento y que estiver após um elemento x x + y{color: violet}
x .class Seleciona o elemento x em que se aplicou a "classe" x .y{color: violet}
x #id Seleciona o elemento x identificado com Id x #id{color: violet}
x,y Seleciona o elemento x e y x,y{color: violet}
x[atributo] Seleciona o elemento x que contenha o atributo indicado x[title]{color: violet}
x[atributo="valor"] Seleciona o elemento x que contenha o atributo e o valor indicado input[type=number]{color: violet}
x:first-child Seleciona o primeiro filho da tag x x:first-child {color: violet}
a:link Aplica-se ao elemento com link ainda não visitado a:link{color: violet}
a:visited Aplica-se ao elemento com link já visitado a:visited{color: violet}
x:active Aplica-se ao elemento x quando este for ativado pelo usuário a:active{color: violet}
x:hover Aplica-se ao elemento x quando o usuário estiver com o cursor sobre o elemento x:hover{color: violet}
x:focus Aplica-se ao elemento x quando o usuário estiver com o cursor precionado x:focus{color: violet}
